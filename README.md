# 三角洲行动 · Delta Force Tactical

一个基于设计文档（`de8941cf0ca54d7519ccc2a3eeaf32d1.jpg`）复刻的 **俯视角战术射击游戏**：7 局 3 胜制攻防爆破对战，完整的经济 / 战备 / 武器 / 护甲 / 医疗系统，3v3 AI 小队对抗。

纯前端实现（原生 JavaScript + HTML5 Canvas），**零依赖、零构建**，双击即玩。

---

## 快速开始

### 方式一：直接双击（最简单）
双击 `index.html`，用现代浏览器（Chrome / Edge / Firefox）打开即可。

### 方式二：本地服务器（推荐，音效更稳定）
```bash
# 在本目录下执行任一命令，然后浏览器打开 http://localhost:8080
py -m http.server 8080          # Windows + Python
python -m http.server 8080      # 通用 Python
npx serve .                     # Node
```
Windows 用户也可直接双击 `启动游戏.bat`。

## 游戏规则（源自设计文档）

- **赛制**：攻防爆破赛，7 局合制，任意一方 **先赢 3 局** 直接夺冠（不会打满 7 局）。
- **单局取胜途径**：
  1. **歼灭** —— 击杀敌方所有存活队员。
  2. **破译** —— 进攻方站点完成密钥破译任务（战术取胜，无需击杀全部敌人）。
  3. **防守超时** —— 防守方在时限内成功阻止破译即可获胜。
- **你的小队（A 队 / 灵萧队）**：3 人小队，进攻方。你操控核心干员，2 名 AI 队友协同。
- **敌方（BRAVO / CHARLIE）**：3 名 AI 防守干员。

## 系统概览

### 经济系统
| 场景 | 奖励 |
|------|------|
| 初始资金 | 800 |
| 小局获胜 | +3000 |
| 小局失败 | +1500 ~ +2900（连败递增） |
| 击杀 | +150（狙击 +100，近战 +250）|
| 破译完成 | +300 |

战备套装分 5 个价位档：`200 / 1000 / 2500 / 4000 / 6000`，共 15 套预设（见下）。

### 武器（`src/data/weapons.js`）
主武器：MK4 冲锋枪、UZI、QBZ95-1、SV-98、M870、勇士 9×19、QJB201、M700、Vector、腾龙、PKM、M14、M250、SR-25。
副武器：G18、沙漠之鹰。近战：黑寡匕首。

### 护甲（`src/data/armor.js`）
头盔：GT1(48) · MC(20) · MHS(30) · DICH(35)。护甲：MK-2(110) · TG(40) · HMP(80) · 武士(80)。

### 医疗 / 道具（`src/data/items.js`）
CAT 止血带、DEK 野战手术包、战地医疗箱、DVE 止疼片、OE2 战斗兴奋剂、高级护甲维修组合。

## 架构

```
三角洲行动/
├── index.html            入口，按序加载脚本，挂载画布与 UI
├── 启动游戏.bat           Windows 本地服务器启动器
├── styles.css            界面样式
├── README.md
├── src/
│   ├── engine/           与游戏无关的通用引擎
│   │   ├── vec2.js         二维向量数学
│   │   ├── rng.js          可复现随机数
│   │   ├── input.js        键鼠输入
│   │   ├── audio.js        WebAudio 程序化音效（无需素材）
│   │   └── particles.js    粒子系统（枪口火花 / 血迹 / 弹壳）
│   ├── data/             纯数据（对应设计文档表格）
│   │   ├── config.js       全局平衡性配置
│   │   ├── weapons.js      全武器数据
│   │   ├── armor.js        全护甲数据
│   │   ├── items.js        全医疗 / 道具数据
│   │   └── loadouts.js     15 套战备预设
│   ├── game/             游戏逻辑
│   │   ├── map.js          地图 / 掩体 / 站点 / 视线遮挡
│   │   ├── combat.js       伤害结算（护甲 / 爆头 / 流血）
│   │   ├── agent.js        干员实体（玩家与 AI 共用）
│   │   ├── ai.js           敌我 AI 行为状态机
│   │   ├── bullet.js       弹道
│   │   ├── economy.js      经济系统
│   │   ├── objective.js    密钥破译目标
│   │   └── game.js         对局状态机 + 主循环 + 渲染
│   └── ui/
│       ├── hud.js          战斗 HUD（血量 / 护甲 / 弹药 / 计分 / 击杀信息）
│       └── buymenu.js      购买阶段界面
├── tests/
│   └── run.js             纯逻辑单元测试（node tests/run.js）
└── de8941cf0ca54d7519ccc2a3eeaf32d1.jpg   原始设计文档
```

代码通过全局命名空间 `DF` 组织（避免 `file://` 下的 ES module CORS 限制，保证双击可玩）。数据与逻辑严格分离——所有平衡性数值集中在 `src/data/`，改数值不动逻辑。
